  4. (30ptos). Resuelva el siguiente problema sin utilizar clases, sólo usando funciones: Considere las ecuaciones de oferta y demanda: $\(S(q) = q^2 + \beta*10\)\( \)\(D(q) = (q-20)^2\)\( El objetivo es crear un gráfico que muestre la demanda y la oferta para 10 valores de \)\beta \in [1,10]\(. En clases vimos que podemos solucionar este equilibrio con `eq = sy.Eq(S(q), D(q))` y `q_sol = sy.solve(eq)`. Ahora, la idea es usar un `for` para iterar sobre 10 valores de \)\beta\( y graficar las ofertas y los puntos de equilibrio (cantidad, precio). Aproveche el `for` para cambiar la intensidad de color de cada curva de oferta (por ejemplo, a menor valor de \)\beta$ menor intensidad de color).

  5. (40ptos). Organice su código del punto 4. utilizando una clase, que contenga al menos:

  • Actualización del parámetro \(\beta\)

  • Función de oferta

  • Función de demanda

  • Función para solución del equilibrio

  • Función para gráfico
